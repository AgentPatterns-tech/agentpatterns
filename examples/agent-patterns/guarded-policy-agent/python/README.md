# Guarded-Policy Agent (Python)

Runnable production-style example of a Guarded-Policy Agent.

## What this shows

- action plan proposal
- strict policy-gate decisions: `allow`, `deny`, `rewrite`, `escalate`
- runtime execution allowlist
- human approval branch for escalations
- transparent `trace` and `history`

## Run

```bash
python main.py
```

No external dependencies are required for this demo.
